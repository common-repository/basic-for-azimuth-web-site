Basic function for your Azimuth web site.

== Description ==

Add basic function.


*   Fix youtube embem code with other content
*   Add in admin panel help widget
*   Change admin panel footer
*   Custom logo for login page
*   Random post function
*   Disable to show error in the log-in page
*   Custom favicon in theme option
*   Stop malicius request
*   Don't show alert of upgrade wordpress


== Installation ==

Just added to your plug-in reposity and then activate.

For show your custom logo put your image file in your templare folder and then image folder and call the image file login_logo.png

For the random post just make a button or similar that link to yourdomain.com/random/ if this doesn't work put yourdomain.com/index.php?random=1

For add your custom favicon go in the theme option section select the file and update.
